# Currency Converter (Python)

A small command-line currency converter that understands inputs like:

- `100 USD to EUR`
- `USD 50 GBP`
- `200,5 JPY to USD`

It parses the text, cleans the number, and converts using a set of predefined rates.

## Features
- Supports a bunch of currencies  
- Flexible input parsing  
- Simple CLI loop (`quit` to exit)

## Run it
```bash
python main.py
